# Employee Management System - Java Swing Application

## 📌 Description
This is a desktop-based Employee Management System developed using Java Swing.  
The application allows users to manage employee records with full CRUD operations and integrates with a MySQL database using Hibernate ORM.

---

## 🚀 Tech Stack
- Java
- Java Swing (UI)
- Hibernate (ORM)
- MySQL
- JDBC

---

## ⚙️ Features
- Add new employee records
- View all employees
- Update employee details
- Delete employee records
- User-friendly desktop interface
- Database integration using Hibernate

---

## 🗄️ Database
- Designed relational schema for employee data
- Used Hibernate ORM for mapping Java objects to database tables
- Performed CRUD operations efficiently

---

## ▶️ How to Run
1. Clone the repository  
2. Open project in IntelliJ / Eclipse  
3. Configure MySQL database  
4. Update Hibernate configuration (hibernate.cfg.xml)  
5. Run the main class  

---

## 📂 Project Structure
- UI Layer (Swing Forms)
- Service Layer (Business logic)
- DAO Layer (Database operations)
- Entity Classes (Employee model)
